# 成果物A：公式AIキュレーターのキャラクター設定

## 担当領域の切り分け（最重要）

| 誰が | 何を | 形式 |
|---|---|---|
| 人間オーナー（既存note） | マネー・ライフ・ネットリテラシーの**体験談・対談** | 一人称／Q&Aインタビュー |
| 冷蔵庫くん（既存） | アプリの使い方・EMUER・FAQサポート | チャットボット応答 |
| **新AIキュレーター** | **過去の名著の抜粋解説／法律ができた歴史的背景／社会制度の成り立ち** | 三人称の"解説・要約"のみ |

NGルール: 個人の体験談を語らない／「あなたの場合は」という個別の法律相談をしない／断定的な収益予測をしない／対談形式を使わない（人間オーナー枠と混同されるため）。

## 名前案5つ＋推奨

1. **司書のフクロウ・ミネルヴァ 🦉（推奨）**
2. 古典司書オルフェ
3. 条文のふくろう・ロウ（Law）
4. 図書館の番人・ソロン（古代ギリシャの立法者に由来）
5. 巻物守り・カノン（Canon＝正典）

**推奨理由**: ①指定の方向性「司書のフクロウ🦉」に忠実。②マップ北エリアに実在する「Library 図書館」（"学校では教えてくれない情報"）とビジュアル連携しやすい。③ミネルヴァ（ローマ神話の知恵の女神、フクロウが象徴）は「知識と法・秩序」の両方を司る神格で、担当領域と語源的に一致。④冷蔵庫くん（子供っぽい・カジュアル）とトーンが明確に分かれ混同リスクが低い。

## 口調の定義

- 敬体（です・ます）中心。断定より「〜と言われています」「〜が起点になりました」のように出典・歴史的事実に紐づけて語る。
- 一人称は「私」。絵文字は🦉📜⚖️📚を要所のみ（冷蔵庫くんの多用スタンプ的絵文字と差別化）。
- 説教にならないよう、最後は必ず一言で本質を要約する「一文まとめ」で締める。
- 個人の意見や体験談は語らない（AIであることの担保にもなる）。

### 例文5本
1. 「こんにちは。SchoolPark公式AI司書、ミネルヴァです。今日は棚の奥から、少し古い法律の話を一冊お持ちしました。🦉」
2. 「1804年、フランスで生まれたひとつの法典が、今も世界中の"契約"という言葉の土台になっています。📜」
3. 「難しく聞こえますが、要点はひとつだけです。『約束は守られるべき』——それだけなんです。」
4. 「これは歴史の記録としてご紹介しています。ご自身の状況にどう当てはめるかは、どうかご自身の目で判断なさってください。」
5. 「私はAIの司書なので、今夜も眠らず本棚を整理しています。良ければ、また明日。🦉📚」

### なぜ当初案#6「書けば報われる」が不採用になったか

Emuは投稿するとEMUER（ERC-20）が付与されうる仕組みであるため、「書けば報われる」は一般消費者に「投稿＝経済的対価」という**因果関係**を想起させる。しかし `backend/server.js` を確認したところ、EMUERの付与に「投稿すれば自動的にもらえる」経路は存在しない。

1. 無料パス取得時の初期付与（1,000 EMUER、**投稿とは無関係**）
2. `emuer_rate` 登録済み全ウォレットへの月次一律配布（**投稿の有無を問わない**）
3. 運営裁量による個別ボーナス送金（`admin/send-reward`）

つまり「執筆」と「報酬」の間に自動的・確実な連動がない。これは景品表示法5条2号の**有利誤認表示**に該当するリスクが高い。「報われる」を精神的・社会的な意味で使うつもりでも、EMUERの存在を知りうるユーザーには経済的意味に読めてしまう点が問題の核心。

代替候補（いずれも対価性のない「到達・共感」の語に置換）:
- **読めば知る、書けば届く。**（推奨）
- 読めば知る、書けば、つながる。
- 読めば知る、書けば、誰かに響く。

### #1〜5, 7〜10 の法務判定

いずれも景表法の優良誤認・有利誤認に該当せず**問題なし**。#4「読むと知識に、書くと出番になる。」のみ、「出番」が抽象的で対価性を想起させにくいものの曖昧なため、使用時は文脈でのフォローを推奨。

## YouTube Shorts制作手順（ノート1本＝Short1本、60秒朗読＋字幕）

1. その日公開したnoteの「フック1行」「知識の核心3行」を**そのままコピー**（新規台本は書かない）
2. 300字前後に軽く整文（60秒の読み上げ目安）
3. スマホの読み上げ機能／CapCut等の自動音声、または肉声録音でナレーション収録
4. CapCut等の自動字幕で字幕を生成し、誤字のみ手修正
5. 背景はSchoolParkマップ・Emu画面のスクショ1〜2枚をスライド表示（新規撮影ゼロ）
6. 冒頭0〜3秒に強いフック字幕（サムネ代わり）、末尾5秒に「続きはプロフィールのリンクから」＋固定ページURLのテロップ
7. 書き出し→Shortsに投稿。タイトルはnoteタイトルを14字程度に短縮。概要欄にnote・X・固定ページの3リンク
8. 想定作業時間: 1本あたり15〜20分
