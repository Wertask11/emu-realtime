💸 ウォレット・NFT
ウォレット ウォレットを接続したのに認証されません ▼
以下の順番でお試しください：
1. ページをリロードしてから再接続
2. MetaMask等のウォレット拡張機能を最新版に更新
3. ブラウザのキャッシュをクリア
4. 別のブラウザで試す（ChromeまたはFirefox推奨）
5. ウォレットアドレスに接続許可が出ているか確認

それでも解決しない場合は「問い合わせ」から連絡ください。
NFT NFTを購入したのに表示されません ▼
NFTの反映には数分〜数十分かかる場合があります。
1. トランザクションハッシュで確定を確認（Polygonscan等）
2. ページを強制リロード（Ctrl+Shift+R）
3. MetaMaskの「NFT」タブで「NFTをインポート」を試す
4. コントラクトアドレスとトークンIDを手動入力

24時間経っても表示されない場合は問い合わせください。TX hashを必ずご用意ください。
ウォレット ガス代って何ですか？いくらかかりますか？ ▼
ガス代とは、ブロックチェーン上の取引処理に必要な手数料です。
SchoolParkはPolygonチェーンを使用しているため、ガス代は非常に安価（数円〜数十円程度）です。
POL（旧MATIC）をウォレットに少量持っておく必要があります。

ガス代ゼロの操作（閲覧・投稿等）と有料操作（NFT購入・トークン送金）があります。
🪙 EMUERトークン
トークン EMUERトークンはどこで手に入りますか？ ▼
EMUERトークン（EMUER）はSchoolPark内の活動を通じて取得できます：
• NFT購入時のボーナス付与
• コミュニティ貢献への報酬
• 特定のコンテンツ購入

詳細はSchoolParkのトークン説明ページをご確認ください。
トークン EMUERトークンが届きません ▼
1. Polygonscanでトランザクションを確認
2. MetaMaskでトークンの「インポート」を行う
  （コントラクトアドレスが必要です）
3. 時間をおいて再確認（ブロック確定まで数分）

24時間以上経過しても届かない場合は問い合わせください。
📝 投稿・コンテンツ
投稿 投稿が反映されません ▼
1. ウォレットが接続されているか確認
2. ページをリロードして再投稿を試みる
3. 文字数・画像サイズの制限を確認（画像は5MB以下推奨）
4. 別のブラウザで試す

※投稿は即時反映される場合と、数分かかる場合があります。
投稿 自分の投稿を削除したい ▼
投稿一覧から対象の投稿を長押し（PCは右クリック）すると削除オプションが表示されます。

機能が見当たらない場合や、緊急で削除が必要な場合は問い合わせフォームからご連絡ください。投稿IDをお知らせください。
🔐 アカウント・ログイン
アカウント Googleログインができません ▼
1. ブラウザのポップアップブロックを無効化
2. シークレットモードではなく通常ウィンドウで試す
3. Googleアカウントのサードパーティアプリ許可を確認
4. 別のGoogleアカウントで試す

解決しない場合は問い合わせから連絡ください。使用ブラウザ・OSをお知らせください。
アカウント ユーザー名を変更したい ▼
現在、ユーザー名の変更機能は開発中です。
変更を希望する場合は問い合わせフォームから以下を記入してお送りください：
• 現在のウォレットアドレスまたはUID
• 変更前・変更後のユーザー名
🖥️ 表示・動作
バグ 画面が真っ白・読み込みが終わらない ▼
1. 強制リロード：Ctrl+Shift+R（Mac: Cmd+Shift+R）
2. キャッシュクリア：ブラウザ設定から「閲覧履歴データを削除」
3. 拡張機能を全て無効にして試す
4. 別のブラウザで確認
5. モバイルでも同じ症状か確認

解決しない場合はブラウザ名・バージョン・OSを添えて問い合わせください。
バグ スマートフォンで表示が崩れる ▼
SchoolParkはスマートフォン対応を進めています。
一部の画面でレイアウトが崩れる場合があります。

症状が強い場合は：
• 機種名・OSバージョン
• 崩れている画面名
• スクリーンショット
を添えて問い合わせフォームからご連絡ください。
